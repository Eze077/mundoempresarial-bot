# Manual de Estilo y SEO - MundoEmpresarial.ar

## Identidad editorial

**MundoEmpresarial.ar** es un medio digital de noticias de negocios orientado a pymes, monotributistas, profesionales independientes y empresarios argentinos. El tono es **informativo, directo y accesible**: explicamos la coyuntura en lenguaje simple para quien toma decisiones de negocio.

---

## 1. Estructura de la nota

Toda nota publicada debe respetar la siguiente estructura:

### 1.1 Bajada / Lead (primer parrafo)

- Va en **negrita**.
- Resume en 1-2 oraciones el hecho central y por que importa al lector pyme.
- Maximo 280 caracteres.
- **Debe contener el keyword de enfoque** (Rank Math: keyword en el primer 10% del contenido, 3 pts).

### 1.2 Subtitulos H2

- El **primer H2** se coloca inmediatamente despues del lead.
- **Debe incluir el keyword de enfoque** (Rank Math: keyword en subheadings, 3 pts).
- Formato recomendado: `"{Keyword}: lo que necesitas saber"` o una frase que resuma la seccion.
- Los **H2 siguientes** deben ser descriptivos del contenido de la seccion, no genericos.
- **Frecuencia:** un H2 cada 2-4 parrafos (300-500 palabras aprox.), nunca mas de 5 parrafos seguidos sin subtitulo.
- **Prohibido:** H2 genericos vacios como "Mas detalles", "En profundidad" o "Contexto" sin relacion con el contenido.

### 1.3 Parrafos

- **Maximo 120 palabras por parrafo** (Rank Math: short paragraphs, 3 pts).
- Ideal: 3-4 oraciones por parrafo.
- Usar **oraciones cortas** (25 palabras maximo ideal).
- Un parrafo = una idea. No mezclar temas.

### 1.4 Citas y declaraciones

- Las declaraciones textuales van en parrafo propio.
- Formato: comillas tipograficas + atribucion.
- Ejemplo: *"Milei ya cumplio. Entro en desgaste", dijo una fuente presente en el evento.*
- Si hay multiples fuentes, separarlas en parrafos distintos.

### 1.5 Datos y cifras

- Los numeros y datos duros van en **negrita** para facilitar el escaneo.
- Ejemplo: *La inflacion de marzo fue del **3,4%**, la mas alta en 6 meses.*
- Si hay 3+ datos, usar **lista con vinetas** en vez de parrafo corrido.

### 1.6 Links

- **Al menos 1 link externo dofollow** (Rank Math: external links, 4 pts + dofollow, 2 pts). El link a la fuente original cumple este requisito.
- **Al menos 1 link interno** a otra nota de MundoEmpresarial.ar (Rank Math: internal links, 5 pts).

### 1.7 Recuadro "Resumen para Pymes"

- Se inserta antes del cierre de la nota.
- Maximo **240 caracteres**.
- Lenguaje ultra simple: que paso y como impacta a una pyme.
- Estilo visual: fondo celeste (#eaf4fb), borde izquierdo azul (#1a6fa8).

### 1.8 Fuente

- Siempre al final de la nota.
- Formato: *Fuente: [Ver nota original](URL)* con `target="_blank" rel="noopener noreferrer"`.
- Este link cuenta como **link externo dofollow** para Rank Math.

## 2. SEO on-page (alineado a Rank Math)

Rank Math evalua cada nota sobre 100 puntos. El objetivo es **80+ (verde)** en cada publicacion. A continuacion los tests agrupados por peso, de mayor a menor impacto.

### 2.1 Keyword de enfoque (Focus Keyword)

La palabra clave es el eje de la optimizacion. Rank Math la evalua en **7 tests distintos** que suman mas de 50 puntos.

| Donde debe aparecer | Test Rank Math | Puntos |
|---------------------|----------------|--------|
| **Titulo SEO** | Keyword in SEO Title | **36** |
| **Primera mitad del titulo** | Keyword at Beginning of Title | **3** |
| **URL / slug** | Keyword in URL | **5** |
| **Primer 10% del contenido** (lead) | Keyword in First 10% | **3** |
| **Cuerpo del articulo** | Keyword in Content | **3** |
| **Al menos un H2** | Keyword in Subheadings | **3** |
| **Alt text de imagen** | Keyword in Image Alt | **2** |
| **Meta description** | Keyword in Meta Description | **2** |

**Reglas para el keyword:**
- Una sola palabra o frase corta.
- Se extrae del titulo: la primera palabra significativa (no stop-word, >3 caracteres).
- **No repetir** el mismo keyword en otra nota (Rank Math: keyword uniqueness).

### 2.2 Densidad del keyword

| Densidad | Rating | Puntos |
|----------|--------|--------|
| 1,0% - 2,5% | Optima | **6** |
| 0,76% - 1,0% | Buena | 3 |
| 0,5% - 0,75% | Aceptable | 2 |
| < 0,5% | Muy baja | 0 |
| > 2,5% | Keyword stuffing | 0 |

Para una nota de 600 palabras con keyword de 1 palabra, la keyword debe aparecer entre **6 y 15 veces**.

### 2.3 Titulo SEO

| Regla | Valor | Pts Rank Math |
|-------|-------|---------------|
| Keyword presente | Obligatorio | **36** |
| Keyword en primera mitad | Primeras 30 chars aprox. | **3** |
| Contiene un numero | Digitos, no letras | **1** |
| Largo maximo | **60 caracteres** (automatico) | - |
| Corte | En limite de palabra, sin puntos suspensivos | - |
| Power words | Palabras de impacto ("clave", "record", "alerta") | **1** |

**Titulo manual vs. automatico:**
- Por defecto el bot recorta el titulo a 60 caracteres (limite SEO).
- Si se usa el boton **"Cambiar titulo"**, el titulo ingresado se publica **tal cual**, sin recorte ni modificacion. El operador es responsable de respetar el largo y las reglas SEO.

**Buenos ejemplos:**
- *AFIP extiende el plazo para monotributistas* (keyword "AFIP" al inicio)
- *Dolar hoy: el blue cerro a $1.450 y el cepo sigue* (numero + keyword "Dolar")
- *Vaca Muerta bate record: 5 claves del boom energetico* (numero + power word)

**Malos ejemplos:**
- ~~Patricia, la favorita entre los empresarios de Amcham: "Milei ya cumplio"~~ (77 chars, keyword no claro)
- ~~IMPORTANTE: Nueva normativa que cambia todo para las pymes~~ (clickbait, keyword al final)

### 2.4 Meta description

| Regla | Valor | Pts Rank Math |
|-------|-------|---------------|
| Keyword presente | Obligatorio | **2** |
| Largo | **120-155 caracteres** (max 156) | - |
| Contenido | Resumir el hecho + gancho para el click | - |

Si el keyword no esta presente en el excerpt original, se antepone: `"Keyword: descripcion..."`.
Corte en limite de palabra + "...".

### 2.5 URL / Slug

| Regla | Valor | Pts Rank Math |
|-------|-------|---------------|
| Keyword presente | Obligatorio | **5** |
| Largo total URL | **<= 75 caracteres** | **4** |
| Formato | Minusculas, sin tildes, guiones | - |

Ejemplo: `afip-extiende-plazo-monotributistas`

### 2.6 Largo del contenido

| Palabras | Puntos Rank Math |
|----------|-----------------|
| 2.500+ | **8** (maximo) |
| 2.000 - 2.499 | 5 |
| 1.500 - 1.999 | 4 |
| 1.000 - 1.499 | 3 |
| 600 - 999 | 2 |
| < 600 | 0 (no pasa) |

**Minimo absoluto: 600 palabras.** Si la nota original es mas corta, se expande con contexto, analisis pyme o datos relacionados.

### 2.7 Links

| Tipo | Requisito | Pts Rank Math |
|------|-----------|---------------|
| Interno | Al menos 1 link a otra nota de MundoEmpresarial.ar | **5** |
| Externo | Al menos 1 link externo | **4** |
| Dofollow externo | Al menos 1 externo sin nofollow | **2** |

El link a la fuente original (al pie) cuenta como externo dofollow.
Los links internos se pueden agregar como "notas relacionadas" o en el cuerpo.

### 2.8 Imagenes y video

| Contenido | Puntos Rank Math |
|-----------|-----------------|
| 0 imagenes | 0 |
| 1 imagen | 1 |
| 2 imagenes | 2 |
| 3+ imagenes | 4 |
| 1+ video embebido | +1 |
| **Maximo** | **6** |

- **Imagen destacada**: Siempre que la fuente la provea (og:image).
- **Alt text**: `"{Keyword} - {Titulo SEO}"`. Ejemplo: *"Inflacion - AFIP extiende plazo monotributistas"*.
- **Alt text debe contener el keyword** (2 pts).

### 2.9 Legibilidad (Readability)

| Test | Criterio | Pts Rank Math |
|------|----------|---------------|
| Parrafos cortos | Todos los parrafos <= 120 palabras | **3** |
| Flesch Reading Ease | Score 60-69 = 4 pts / 70-79 = 5 pts / 80+ = 5-6 pts | **hasta 6** |
| Tabla de contenidos | TOC presente en la nota | **2** |

**Para mejorar Flesch:**
- Oraciones cortas (max 25 palabras).
- Palabras simples (evitar jerga tecnica sin explicacion).
- Voz activa ("AFIP extiende el plazo" en vez de "El plazo fue extendido por AFIP").

## 5. Etiquetas (Tags)

- Hasta **8 etiquetas** por nota.
- Se extraen del titulo + primer parrafo del contenido.
- Minimo 4 caracteres, sin stop-words.
- Cada etiqueta va en **Title Case** (primera letra mayuscula).
- Sin duplicados.

## 6. Redes sociales

### 6.1 Twitter / X

| Elemento | Regla |
|----------|-------|
| Largo total | **280 caracteres** max |
| Formato | Titulo + URL + Hashtags |
| Titulo | Usar titulo SEO (60 chars max) |
| Hashtags | Hasta 3 del titulo + #Pymes fijo |
| URL | Link al post en WordPress |

Estructura:
```
{Titulo}

{URL WordPress}

#{Tag1} #{Tag2} #{Tag3} #Pymes
```

### 6.2 Canal de Telegram (@MundoEmpresarial_AR)

Estructura:
```
📰 **{Titulo}**

{Excerpt, 200 chars max}

🔗 Leer nota completa: {URL}
```

Con imagen destacada cuando este disponible.

### 6.3 WhatsApp (copy-paste manual)

Estructura:
```
📰 {Titulo}

{Excerpt, 200 chars max}

🔗 {URL}
```

## 7. Limpieza de contenido scrapeado

El bot automaticamente elimina del texto:

- Fragmentos de UI: "compartir en", "seguinos", "suscribite", etc.
- Secciones de comentarios
- Mensajes de error de browser/HTML5
- Lineas con encoding roto (caracteres Ã, Â, â€, etc.)
- Lineas menores a 25 caracteres que no terminan en puntuacion

## 9. Checklist de publicacion

### SEO Basico (57 pts)
- [ ] Keyword en titulo SEO, en la **primera mitad** (36 + 3 pts)
- [ ] Keyword en URL/slug, URL <= 75 caracteres (5 + 4 pts)
- [ ] Keyword en primer parrafo / lead (3 pts)
- [ ] Keyword presente en el cuerpo (3 pts)
- [ ] Meta description 120-155 chars con keyword (2 pts)
- [ ] Contenido >= 600 palabras (8 pts min 2, ideal 2500+ para 8)

### SEO Adicional (22 pts)
- [ ] Keyword en al menos un H2 (3 pts)
- [ ] Keyword en alt text de imagen (2 pts)
- [ ] Densidad del keyword 1-2,5% (6 pts)
- [ ] Al menos 1 link interno a otra nota (5 pts)
- [ ] Al menos 1 link externo dofollow / fuente (4 + 2 pts)

### Legibilidad (17 pts)
- [ ] Todos los parrafos <= 120 palabras (3 pts)
- [ ] Imagenes: al menos 1, ideal 3+ (hasta 4 pts)
- [ ] Flesch Reading Ease >= 60 (hasta 6 pts)
- [ ] Tabla de contenidos presente (2 pts)

### Titulo (2 pts extra)
- [ ] Numero en el titulo cuando aplique (1 pt)
- [ ] Power word en el titulo (1 pt)

### Editorial
- [ ] H2 descriptivos, no genericos
- [ ] Citas en parrafo propio con atribucion
- [ ] Datos y cifras en negrita
- [ ] Recuadro "Resumen para Pymes" presente
- [ ] Fuente con link al original
- [ ] Categorias correctas (max 3)
- [ ] Etiquetas relevantes (max 8)
